## 🏗️ Tech Stack That Powers the Magic

| Layer | Technology | Why We Chose It |
|-------|------------|----------------|
| 🎨 **Frontend** | Next.js 14 + TypeScript | Blazing fast, type-safe, and sexy |
| ⚡ **Backend** | Express.js + Node.js | Reliable, scalable, battle-tested |
| ☁️ **Serverless** | AWS Lambda | Auto-scaling for viral stories |
| 🗄️ **Database** | MongoDB Atlas + Vector Search | Semantic story understanding |
| 🤖 **AI Engine** | Gemini API + RAG | Actually intelligent story assistance |
| 🔐 **Auth** | NextAuth.js + JWT | Secure and seamless |
| 🎨 **Styling** | Tailwind CSS + Framer Motion | Beautiful and buttery smooth |

## 🚀 Quick Start - Join the Chaos!

### 🔥 **Option 1: Just Try It (Recommended)**
[![Try StoryTime.AI](https://img.shields.io/badge/🎭_Try_StoryTime.AI-No_Setup_Required-green?style=for-the-badge)](https://story-time-ai-2-o.vercel.app/)

### 💻 **Option 2: Run Locally (For Developers)**

```bash
# 1. Clone the madness
git clone https://github.com/Rajukrsna/StoryTimeAI-2.o.git
cd StoryTimeAI-2.o

# 2. Frontend setup
cd StoryTime-Frontend
npm install
npm run dev

# 3. Backend setup (new terminal)
cd ../story_backend
npm install
npx nodemon server.js

# 4. Visit http://localhost:3000 and start creating chaos! 🎉
```

## 🔐 Environment Setup

### **Frontend (.env.local)**
```env
NEXT_PUBLIC_API_URL=http://localhost:5000
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-super-secret-key
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret
```

### **Backend (.env)**
```env
MONGODB_URI=your-mongodb-connection-string
GEMINI_API_KEY=your-gemini-api-key
JWT_SECRET=your-jwt-secret
FRONTEND_URL=http://localhost:3000
NODE_ENV=development
```
